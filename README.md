# Timberborn Seasons

Seasons is a mod that adds a season cycle and weather events to the game Timberborn.

### Features
 - Calender system
 - Cycle between the 4 seasons (spring, summer, autumn, winter)
 - Each season has it's own visual style
 - Weather events:
   - Rain
   - Wind
   - Drought
   - Flood
   - Sunny
